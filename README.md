# UI/UX Prototypes

Repositorio de prototipos interactivos, exploraciones visuales y handoff UI/UX.

Cada carpeta dentro de `prototypes/` contiene un entregable navegable. La raíz funciona como índice público para GitHub Pages.

## Convención

- `prototypes/<slug>/index.html`: prototipo principal.
- `prototypes/<slug>/notes.md`: intención, decisiones y handoff cuando aplique.
- Evitar assets externos frágiles salvo que el proyecto lo requiera.
